# Commercialization Notes

NovelTips is designed as a vertical narrative agent workspace, not a generic chatbot.

## Product Tiers

### Creator

- chapter continuity check
- character card generation
- setting query
- lightweight scene simulation

### Studio

- shared world graph
- full Character Skill library
- plot branch traces
- editor review reports

### Platform / IP

- authorized IP character interaction
- reusable role packages
- cross-work setting management
- reader feedback analysis
- adaptation pre-production support

## Commercial Differentiation

Compared with a long-context chatbot, NovelTips stores durable narrative state:

- world facts
- character permissions
- event timeline
- scene state
- review reports
- replayable traces

Compared with a pure writing assistant, NovelTips focuses on long-form consistency and interactive IP operations.

## Roadmap

1. Pi extension registration and slash commands
2. Neo4j adapter replacing JSON world store
3. SQLite scene state store
4. model-router integration
5. reader feedback classifier
6. studio workspace UI
7. authorization and audit layer for commercial IP
